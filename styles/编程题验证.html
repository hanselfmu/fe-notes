简易运算器
时间限制：C/C++ 1秒，其他语言2秒
空间限制：C/C++ 32.000MB，其他语言64.000MB

题目描述

实现一个简单的运算器。接收的代码包括以下四种：
1. 变量定义；
2. 变量赋值；
3. 加减乘除运算；
4. return一个值

代码不包含任何函数定义或调用，不包含条件判断或循环。事实上，代码片段仅会有如下关键词：
1. var
2. return
代码中仅会出现如下操作符："+"， "-"， "*"， "/"，并且仅作为二元运算子使用。代码中不会出现比较操作符（"==", "==="等）。 代码中只会出现两种JavaScript基础数据类型：Number和String。

可以做如下假设：
1. 代码片段的语法是正确的，不需要做语法检查；
2. 代码由";"分割；
3. 代码片段对上下文或宿主环境没有依赖，不会调用process, window等特殊变量。

输入描述:

包含代码片段的字符串。

输出描述:

代码片段之行后return的结果。
示例1

输入

"var a = 1; var b = 2; return a + b;"

输出

3
示例2

输入

"var a = 'abc'; var b = 'de'; var c = a + b; return c + a;"

输出

abcdeabc
示例3

输入

"var a = 3; var b = 'a'; return a / b + 1;"

输出

NaN
示例4

输入

"var a = 5; var b = 7; return 42;"

输出

42
